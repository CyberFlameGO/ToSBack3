ToSBack 3

ToSBack 3 is a terms-of-service and privacy-policies crawler, archiver, and diff tracker written in Ruby on Rails.

Download .zip Download .tar.gz View on GitHub

What is ToSBack?

ToSBack was created to help you monitor the policies and terms-of-servuce for the websites you use everyday, and show how they change over time.

ToSBack 3 is an interactive web application built on Ruby on Rails.

How you can help!

  • If you aren't a developer, help us add policies during the hackathon.
  • Check the issues page or this wiki article to find a task you want to work on!
  • Here is a good guide for creating an issue in the tracker, forking, branching, and submitting a pull request!
